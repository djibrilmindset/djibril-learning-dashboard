## MATRICE TOFU / MOFU / BOFU

### TOFU (60% du volume) — "JE TE VOIS"
**Objectif** : Constat + Voyage dans la douleur. PAS de solution.
**Règle** : Reconnaître. Pas guérir.

```
Hook     → Constat qui fait mal (1 phrase)
Connexion→ "T'es pas seul dans ce cas"
Solution → RIEN (pas encore)
Métaphore→ Image du quotidien barber
Micro-action → RIEN (pas encore)
CTA      → 🔔 (Abonne-toi)
```

**Exemple TOFU** :
> "60h/semaine. 6 jours sur 7. Et à la fin du mois t'as les mêmes fins qu'au début." (Hook)
> "C'est pas toi. C'est le système." (Connexion)
> [Pas de solution — juste la reconnaissance]

### MOFU (30% du volume) — "JE T'ÉDUQUE"
**Objectif** : Éducation + Action CE SOIR. Framework + micro-action.
**Règle** : UN concept. UNE micro-action applicable ce soir.

```
Hook     → Question qui challenge
Connexion→ 1 phrase qui valide sa frustration
Solution → LE concept / framework (1 seule idée)
Métaphore→ Image qui ancre le concept
Micro-action → 1 chose à faire CE SOIR (spécifique)
CTA      → 📌 (Enregistre)
```

**Exemple MOFU** :
> "Pourquoi certains barbers gagnent 10k€/mois sans bouger leur agenda ?" (Hook)
> "Parce qu'ils ont arrêté de vendre leurs heures. Ils vendent du résultat." (Connexion)
> [Framework : Arrête de vendre du temps. Vend de la transformation.] (Solution)
> "Ce soir : note les 3 choses que tes clients OBTIENNENT en sortant de ton shop." (Micro-action)
> 📌

### BOFU (10% du volume) — "JE TE CLOSE"
**Objectif** : Conversion. Offre. Garantie. Close.
**Règle** : Plus long à écrire. Plus de valeur. Garantie en dernier.

```
Hook     → Stat ou résultat qui attise la curiosité
Connexion→ "Ça s'appelle [nom méthode], et ça a changé [nb] barbers"
Solution → L'offre (brièvement — prix, durée, promesse)
Garantie → 10k€ générés SINON refund + 1000€ (contrat avocat)
CTA      → 🎯 (AUDIT)
```

## CRITÈRES DE VALIDATION — LE GÉNÉRAL

Tout script doit passer ces 6 critères. **1 seul NON = RENVOYÉ.**

| # | Critère | Question |
|---|---------|----------|
| 1 | **Hook Power** | Le hook arrête le scroll en 1.5s ? |
| 2 | **Clarté** | L'idée est comprise en 3s ? |
| 3 | **Actionnabilité** | Solution actionnable CE SOIR (MOFU/BOFU) ? |
| 4 | **Empathie** | Le prospect se sent VU, pas jugé ? |
| 5 | **CTA clair** | Le CTA a une raison EXPLICITE ? |
| 6 | **Différenciation** | C'est mieux que ce que les concurrents font ? |

**Format de sortie attendu par Le Général** :
```
VALIDÉ ✅
ou
RENVOYÉ ❌ — [Critère échoué] — [Correction précise]
```

## RÈGLES ABSOLUES DE GÉNÉRATION

1. **30 secondes pile** — Pas 31, pas 29. Pile.
2. **1 idée = 1 script** — Jamais 2 concepts dans 1 script.
3. **Tutoiement systématique** — Tu / toi / ton / tes.
4. **0 faute** — Orthographe, grammaire, accords.
5. **0 label visible** — Pas de "Hook:", "Solution:", "Cadre:".
6. **Pas de handles (@)** — Jamais.
7. **Pas de hashtags** — Dans le corps du script.
8. **Emojis** — Uniquement dans le CTA.
9. **Pas de question rhétorique en hook** — Juste une affirmation choc.
10. **Pas de "savais-tu que"** — Interdit.
